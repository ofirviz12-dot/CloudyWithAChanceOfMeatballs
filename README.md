## üöÄ Project Overview
This is an Android arcade game project titled **"Cloudy with a Chance of Meatballs."**  
The central objective is for the character Flint to continuously dodge the incoming meatballs.  
The application was developed in **Kotlin** using **Android Studio** as a practical assignment for a **User Interface (UI) Development** course.

**Game Modes:**  
The game includes 3 modes:
- Slow Mode  
- Fast Mode  
- Tilt Mode (gameplay using device tilt)

## ‚ú® Key Features
- Collision Detection: Precise handling of player movement and collision with falling objects (meatballs).
- Highscore Table: Displays the best scores achieved in the game.
- Trophy Icon: A trophy icon that navigates directly to the highscore table.
- Sensors Integration: Gameplay using device sensors (tilt control).
- Sound Effects: Sound is played on collision or coin collection.
- Haptic Feedback: Vibration on collision.
